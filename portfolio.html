Programa de Gerenciamento de Lojas

Desenvolvi um programa para a empresa onde trabalho que realiza as seguintes funções:

  • Cria um banco de dados a partir de uma planilha fornecida.
  • Permite a entrada do número de uma loja e retorna as seguintes informações:
    • Grupo da Loja
    • Versão do programa a ser utilizado
    • IP do servidor da loja
  • Gera um arquivo .cfg com o nome da loja, que pode ser utilizado para operar o software da empresa de maneira adequada.
